# TMA Risk Prediction Shiny App

## Overview
This project develops an interactive Shiny application to explore and model risk factors for Transplant-Associated Thrombotic Microangiopathy (TMA).

## Features
- Exploratory Data Analysis
- Boruta Feature Selection
- Logistic Regression Modeling
- Risk Stratification
- Survival Analysis

## Tools Used
- R
- Shiny
- Boruta
- Survival Analysis

## Dataset
RT14-02.sas7bdat
